# Shared model assets

This folder contains assets shared across many models, organized by backend.

## Structure

- `models/_shared/<backend>/templates/`: shared template experiments/overrides
- `models/_shared/<backend>/prompts/`: shared prompt assets (system prompts, few-shots, tool schemas)

Per-model assets live under:
- `models/<model>/<backend>/notes/`
- `models/<model>/<backend>/config/`
- `models/<model>/<backend>/templates/`
- `models/<model>/<backend>/prompts/`
